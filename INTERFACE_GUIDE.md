## 📊 1. 仪表板数据区（顶部）

### 四个关键指标

```
┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐
│  资源总数        │  │  低库存资源      │  │  预警总数        │  │  流水总数        │
│      15          │  │       2          │  │       5          │  │       48         │
└──────────────────┘  └──────────────────┘  └──────────────────┘  └──────────────────┘
```

- **资源总数**：共享资源总数（设备 + 物料）
- **低库存资源**：数量低于阈值的资源个数（需要补货）
- **预警总数**：系统生成的预警数量（库存不足、占用不均、疑似浪费）
- **流水总数**：所有交易记录数量（借还、领用、补货等）

### 表格结构

| ID | 名称 | 类型 | 可用/总量 | 位置 | 操作 |
|----|------|------|----------|------|------|
| 1 | 3D打印机 | device | 1/3 | 305室 | [查看] |
| 2 | 激光切割机 | device | 0/2 | 307室 | [查看] |
| 3 | 焊锡 | material | 5/100 | 耗材柜 | [查看] |
| ... | ... | ... | ... | ... | ... |

### 列说明

- **ID**：资源的唯一标识符
- **名称**：资源的友好名称
- **类型**：device（设备）或 material（物料）
  - 🔧 设备：3D打印机、激光切割机、电子元器件套件等（可借用）
  - 📦 物料：焊锡、热缩管、3D耗材等（一次性消耗品）
- **可用/总量**：当前可用数量 / 总库存量
  - 数字变红 = 库存不足
  - 数字变绿 = 库存充足
- **位置**：存放地点（如：305室、耗材柜）

## 🔄 3. 操作面板（右侧）

### 3.1 借用资源（设备）

**适用对象**：3D打印机、激光切割机等**设备类**资源

**步骤**：
1. 在"借用资源"标签输入
2. 选择资源：下拉菜单选择一个设备
3. 输入时间：
   - 预计借用时间（日期 + 时间）
   - 预计归还时间（日期 + 时间）
4. 输入目的：描述为何需要该设备
5. 点击"提交借用"

**系统检测**：
- ✅ 自动检测时间冲突（该设备在此时间段是否已被借用）
- ✅ 无冲突 → 申请成功，进入待审批或已批准状态
- ❌ 有冲突 → 提示"时段已被占用，请选择其他时间"

**表单示例**：
```
┌─────────────────────────────────┐
│ 借用资源                         │

│ 选择资源：[3D打印机 ▼]          │
│                                 │
│ 借用时间：[2024-03-24] [14:00] │
│ 归还时间：[2024-03-24] [16:00] │
│                                 │
│ 使用目的：                      │

│                                 │
│              [提交借用]          │
│                                 │
└─────────────────────────────────┘
```

### 3.2 领用物料

**适用对象**：焊锡、热缩管、3D耗材等**物料类**资源

**步骤**：
1. 在"领用物料"标签输入
2. 选择物料：下拉菜单选择一种物料
3. 输入数量：需要领用多少个/克/卷等
4. 输入用途：描述具体用途
5. 点击"提交领用"

**审批规则**：
- 📝 数量 < 10：自动批准
- 🔏 数量 ≥ 10：需要管理员审批
- 📊 拒绝原因：库存不足或超过单次领取限额

## 🎯 常见操作流程

### 流程 1：学生借用设备并等待审批

```
1. 登录 (student001)
   ↓
2. 点击"借用资源"标签
   ↓
3. 选择"3D打印机" → 输入时间 → 输入目的
   ↓
4. 点击"提交借用"
   ↓
5. 系统检查时间冲突 ✓
   ↓
6. 查看状态："待批准" ⏳
   ↓
7. （可选）用 AI 智能体查询："我的申请批准了吗？"
   ↓
8. 管理员批准 ✓
   ↓
9. 状态变更为"已批准" ✓
```

### 流程 2：管理员审批申请

```
1. 登录 (admin001)
   ↓
2. 点击"审批管理"标签
   ↓
3. 查看"待审批 (3)"列表
   ↓
4. 逐项阅读申请详情
   ↓
5. 填写审批意见（可选）
   ↓
6. 点击"✓ 批准"或"✗ 拒绝"
   ↓
7. 申请状态更新 ✓
   ↓
8. 用户收到通知（下次登录可见）
```

### 流程 3：用 AI 助手快速查询

```
1. 点击底部聊天框
   ↓
2. 输入自然语言问题
   例："明天下午2点有3D打印机吗？"
   ↓
3. AI 秒速回复

    明天 14:00-16:00 可用。点击这里借用。"
   ↓
4. 一键跳转借用表单
```
