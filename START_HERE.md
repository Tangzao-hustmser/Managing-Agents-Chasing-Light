## 🔧 常见操作

### 停止服务
```bash
# 在终端按 Ctrl+C
```

### 重启服务（如修改代码）
```bash
# 会自动重启（热加载已启用）
# 或手动：Ctrl+C 后重新运行
python -m uvicorn app.main:app --reload
```

### 重置数据库
```bash
# 删除数据库文件
del smart_lab.db

# 重新初始化
python -m app.seed

# 重启服务
python -m uvicorn app.main:app --reload
```

### 查看数据库内容
```bash
# 使用 sqlite3 命令行
sqlite3 smart_lab.db

# 查看所有表
.tables

# 查询用户
SELECT * FROM users;

# 查询资源
SELECT id, name, available_count, unit_cost FROM resources;
```

## ❓ 常见问题

### Q: 看不到界面
A: 确保：
- 终端显示 `Application startup complete`
- 浏览器访问 `http://127.0.0.1:8000/dashboard`
- 不是 `https://`（应该是 `http://`）

### Q: 登录失败
A: 检查：
- 用户名和密码是否正确（上面的表格）
- 数据库是否已初始化：`python -m app.seed`

### Q: 报错"连接被拒绝"
A: 检查：
- 服务是否启动（终端是否显示 `Uvicorn running on`）
- 是否访问了正确的端口（`:8000`）

### Q: 想修改前端
A: 前端文件位置：
```
app/static/dashboard.html
```
编辑后刷新浏览器即可看到变化（由于热加载，可能需要清除浏览器缓存）
