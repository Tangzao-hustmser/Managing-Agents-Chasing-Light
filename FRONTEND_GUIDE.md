# 前端界面访问指南

## 🌐 访问方式

当应用启动后（`python -m uvicorn app.main:app --reload`），可通过以下方式访问前端：

### 1. **智能管理面板**（推荐）
```
http://127.0.0.1:8000/dashboard
```
- 完整的资源管理和智能体交互界面
- 支持设备借还、物料领用、审批管理
- 实时库存预警和数据分析

### 2. **API 交互界面**（Swagger）
```
http://127.0.0.1:8000/docs
```
- 所有 REST API 的在线测试工具
- 可直接调用接口，查看请求/响应
- 包含完整的参数说明和返回值文档

### 3. **API 替代文档**（ReDoc）
```
http://127.0.0.1:8000/redoc
```
- 更清晰的 API 文档格式
- 按模块分类展示所有端点

### 4. **首页**
```
http://127.0.0.1:8000/
```
- 跳转到 `/dashboard`

### 登录方式

前端界面的登录步骤：
1. 访问 `/dashboard`
2. 点击"登录"按钮
3. 输入用户名和密码
4. 点击"确认登录"

使用 API 的登录：
```bash
curl -X POST http://127.0.0.1:8000/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin001", "password": "admin@2024"}'
```

响应示例：
```json
{
  "user": {
    "id": 1,
    "username": "admin001",
    "real_name": "管理员",
    "student_id": "2024000001",
    "role": "admin"
  },
  "token": "Bearer 1"

```

## 📋 常见操作示例

### 场景 1：学生借用 3D 打印机

1. **登录**：使用 `student001` / `student@2024`
2. **导航**：点击"资源中心"→"设备"

## 🐛 故障排查

### 前端加载失败

**症状**：访问 `/dashboard` 显示空白或 404

**原因和解决**：
1. 检查服务是否启动：`http://127.0.0.1:8000` 是否响应
2. 检查静态文件是否存在：`app/static/dashboard.html`
3. 检查 `main.py` 中的 StaticFiles 挂载是否正确

```python
app.mount("/static", StaticFiles(directory="app/static"), name="static")
```

### 登录失败

**症状**：输入账号密码后提示错误

**原因和解决**：
1. 检查密码是否正确（区分大小写）
2. 确保数据库已初始化：`python -m app.seed`
3. 查看服务器日志是否有错误

### API 请求超时

**症状**：操作时显示"请求超时"

**原因和解决**：
1. 检查 LLM 配置（如果用到智能体）：`.env` 中 `LLM_BASE_URL` 是否配置
2. 检查网络连接（如果调用外部 API）
3. 检查数据库是否响应：`python -c "from app.database import engine; print(engine.execute('SELECT 1'))"`

## 📚 API 文档快速导航

### 认证 API
- `POST /auth/register` - 用户注册
- `POST /auth/login` - 用户登录
- `GET /auth/me` - 获取当前用户信息

### 资源 API
- `GET /resources/` - 列表资源
- `GET /resources/{id}` - 获取单个资源
- `POST /resources/` - 创建资源（仅管理员）
- `PUT /resources/{id}` - 更新资源（仅管理员）

### 交易 API
- `GET /transactions/` - 获取交易记录
- `POST /transactions/` - 创建新交易（借用/领用等）
- `PUT /transactions/{id}` - 更新交易（归还等）

### 审批 API
- `GET /approvals/` - 获取待审批列表
- `POST /approvals/{id}/approve` - 批准申请
- `POST /approvals/{id}/reject` - 拒绝申请

### AI 智能体 API
- `POST /agent/chat` - 对话入口
- `GET /agent/sessions` - 获取会话列表
- `POST /agent/chat` - 与智能体对话

完整 API 文档：访问 `http://127.0.0.1:8000/docs`

## 💡 开发相关

### 前端文件位置
```
app/static/
└── dashboard.html  (单文件前端应用，包含 HTML + CSS + JavaScript)
```

### 修改前端

如果需要修改 dashboard.html，编辑后刷新浏览器即可看到变化（由于使用 `--reload` 参数，也可能需要重启服务）。

### 扩展前端

如果要用 Vue/React 等框架重写前端：
1. 在 `app/static/` 下建立新的项目目录
2. 修改 `app/main.py` 的静态文件挂载指向
3. 更新 `/dashboard` 路由返回新的 HTML 文件

## 📞 技术支持

遇到问题可以：

1. **查看服务器日志**：Uvicorn 的实时输出
2. **检查浏览器控制台**：F12 → Console 标签查看 JavaScript 错误
3. **使用 Swagger**：`/docs` 测试 API 是否正常
4. **重启服务**：`Ctrl+C` 停止，重新启动 `python -m uvicorn app.main:app --reload`
